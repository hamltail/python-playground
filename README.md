# python-playground 🐍

Python学習用の実験プロジェクトです。  
仮想環境（venv）や pip、Git など、Python開発の基礎を整えるための練習場として使っています。

---

## ✅ このプロジェクトでやってること

- 仮想環境（venv）の構築
- pipでライブラリを管理
- requirements.txtによる環境の記録
- Gitによるバージョン管理とGitHubへのPush

---

## 🔧 セットアップ方法

```bash
# 仮想環境を作成（初回のみ）
python -m venv venv

# 仮想環境を有効化
source venv/bin/activate

# ライブラリをインストール
pip install -r requirements.txt
